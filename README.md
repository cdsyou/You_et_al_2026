# You_et_al_2026

For codes that relate to You et al, 2016 (Cell Reports):

"bursting_stardist_laptrack" is the main file I used to execute the analysis. 
"analyse_burst_mvs" contains the workflow and some functions. 
"spot_detection_functions" contains functions that support both the "analyse_burst_mvs" and "bursting_stardist_laptrack" scripts.
"process_periods" is used to process output

"examine" is used to visualise the output for manual verification
